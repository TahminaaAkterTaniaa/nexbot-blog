# Blog Specification

## Features
1. Home page with blog post list
2. Individual post pages
3. About page
4. Contact form

## User Stories
- As a reader, I want to browse articles
- As a reader, I want to read full articles
- As an admin, I want to publish content

## Acceptance Criteria
- [ ] Home page loads in < 2 seconds
- [ ] All links work correctly
- [ ] Mobile responsive
